# n8n Integrations Fetcher

A tool that retrieves the current list of supported integrations from n8n.io and displays them in an alphabetically sorted format.

## Features

- Fetches live integration data from n8n.io
- Displays results as a clean, sortable list
- Alphabetically organized integrations
- Lightweight and fast

## Requirements

- Selenium
- Chrome Browser Extension

## Usage

```bash
python3 nodes.py
